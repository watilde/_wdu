wdu
===

npm clone can use json5

## Install

Using npm.

```
$ npm install -g wdu
```

Then you can write package.json in json5

```
// This file is written in JSON5 syntax
{
    name: 'this',
    description: 'file',
    author: 'has <filename>',
    version: '0.0.1'
}
```
